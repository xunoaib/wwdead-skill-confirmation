# WWDead Skill Purchase Confirmation

This Greasemonkey/Tampermonkey userscript adds confirmation dialogs to all skill purchase buttons on [wwdead.com/classic/skills](https://www.wwdead.com/classic/skills) to prevent accidental skill purchases.

**[Click here to install](https://raw.githubusercontent.com/xunoaib/wwdead-skill-confirmation/main/wwdead-skill-confirmation.user.js)**
