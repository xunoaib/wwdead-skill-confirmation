# WWDead Skill Purchase Confirmation

This Greasemonkey/Tampermonkey userscript adds confirmation dialogs to all `Buy` buttons on the [WWDead skills](https://wwdead.com/classic/skills) page to prevent accidental skill purchases (especially Brain Rot!)

**⚡ [Click here to install](https://raw.githubusercontent.com/xunoaib/wwdead-skill-confirmation/main/wwdead-skill-confirmation.user.js) ⚡**
